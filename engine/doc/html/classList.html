Detailed Description

Gestisce una lista di nodi e le loro matrici di trasformazione per il rendering.

La classe List mantiene una lista di nodi e le relative matrici di trasformazione che sono utilizzate per il rendering. La lista viene utilizzata per ordinare e gestire il rendering degli oggetti nella scena.
