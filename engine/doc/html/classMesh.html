Detailed Description

Rappresenta una mesh che viene renderizzata all'interno di una scena.

Una mesh è una rappresentazione tridimensionale di una forma. Include dati geometrici, come vertici e facce, ed è associata a un materiale per definire l'aspetto visivo della superficie. Le mesh possono anche proiettare ombre.
